# log 日志打印

这是 log 的 flag 设置，flag 就是前缀的那些信息。

结果：

log.SetFlags(log.Ldate|log.Ltime | log.LUTC|log.Llongfile)

log.SetPrefix("Mylog ") 就是设置前缀。

这个 flag 就是对应着二进制位数，用 | 或方法来相加。

log 分为三种方法：print、panic、Fatal

print : 打印到日志输入。

panic : 触发 panic

fatal ：打印完毕后立即退出。
