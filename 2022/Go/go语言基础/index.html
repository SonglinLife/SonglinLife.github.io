# 切片

注意切片

结果：

拥有相同的地址空间，也就是说，切片实际上只是一个引用。

即使用 make 指定了切片的大小，它也可以扩容。

要从切片 a 中删除索引为 index 的元素，操作方法是 a = append(a[:index], a[index+1:]...)

用 copy 作为切片的拷贝，如果直接用 = 则是引用。
