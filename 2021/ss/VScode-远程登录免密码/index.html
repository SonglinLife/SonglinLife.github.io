这个 VScode 免密码登录有点搞，参考了好几个文章，发现都没写全。

VSCode 远程开发 (带免密) 这篇文章大致写的还可以

# remote 机生成 ssh keys

ssh-keygen

# local 机生成 ssh keys

同样，这里，拷贝 local 机的 id_rsa.pub 秘钥内容

# 登录 remote 机

echo "local机的pub秘钥" >>authorized_keys

chmod 600 authorized_keys （没加这个命令死活登录不上）

vscode ssh
